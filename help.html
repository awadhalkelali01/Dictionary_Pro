مركز المساعدة

هنا ستجد شرحاً مبسطاً يساعدك على استخدام القاموس بطريقة صحيحة وسريعة.

البدء السريع

  1. اذهب إلى صفحة "الطلاب" وأضف الطالب وأدخل اسمه وصفه.
  2. اختر الطالب من قائمة الطلاب في صفحة القاموس.
  3. ابدأ بإضافة الكلمات والجمل مباشرة.
  4. استخدم البحث والتصنيفات لتسهيل الوصول للكلمات.
  5. افتح أي كلمة لعرض تفاصيلها أو تعديلها أو حذفها.

إدارة البيانات

  • يمكنك تصدير قاموس الطالب إلى ملف JSON أو XML.
  • يمكنك استيراد ملف يحتوي على كلمات جاهزة.
  • يتم تجاهل الكلمات المكررة تلقائياً.
  • بيانات كل طالب محفوظة داخل جهازك فقط.

النطق الصوتي

داخل بطاقة الكلمة، يمكنك نطق الكلمة والجملة باللهجات الإنجليزية المختلفة، مع القدرة على التحكم في سرعة النطق.

الاختبارات

يمكنك استخدام صفحة الاختبارات لإنشاء تدريبات للطالب، مثل:

  • اختيار من متعدد.
  • كتابة الكلمة (Spelling).
  • تكملة الجملة.

أسئلة شائعة

  • هل يتم حفظ البيانات تلقائيًا؟ نعم، كل كلمة أو تعديل يتم حفظه فورًا.
  • هل تضيع البيانات عند تحديث المتصفح؟ لا، طالما لم يتم مسح التخزين المحلي.
  • هل يمكن استخدام القاموس بدون إنترنت؟ نعم، يعمل محليًا بالكامل.
  • هل يمكن مشاركة القاموس بين عدة أجهزة؟ نعم عبر التصدير والاستيراد.
قاموس الطالب أداة تعليمية سهلة للمنزل والطلاب.
